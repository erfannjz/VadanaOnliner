## 🔹طریقه نصب
کافی است بعد از دانلود پروژه و یا Clone کردن آن، در محلی که فایل `requirements.txt` قرار دارد، از دستور زیر استفاده کنید تا کتابخانه های مورد نیاز نصب شوند.  
```bash
$ pip install -r requirements.txt
```
